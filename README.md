- [x] Create account

- [ ] Follow User
- [ ] Unfollow User
- [ ] Edit my profile
- [ ] See user profile
- [ ] See the full photo
- [ ] Upload a photo
- [ ] Edit the photo (Delete)
- [ ] See the feed
